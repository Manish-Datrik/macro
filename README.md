# TravelPad

travelPad is a small 4 key macropad that i have built . I wanted to create something that connects with my interest in travel, so I designed this macropad to make common travel-related tasks quicker and easier.

The idea is that instead of opening different websites manually, I can press a single key to launch things like Google Maps, flight booking sites, hotel booking websites, or my travel planner.

I designed the PCB in KiCad and created the enclosure in Fusion 360. The macropad uses a Seeed Studio XIAO RP2040 and includes a 0.91-inch OLED display.

---

## Why I Chose This Theme

I enjoy travelling and wanted my project to say. Instead of making a regular macropad, I decided to give it a travel theme with custom artwork and a layout inspired by planning trips.

The goal was to build something that is both useful and personal.

---

## Features

- 4 programmable mechanical keys
- Seeed Studio XIAO RP2040
- 0.91-inch OLED display
- USB-C connection
- Custom PCB designed in KiCad

---

## Example Shortcuts

Some example shortcuts I plan to use are:

- Open Google Maps
- Open flight booking websites
- Open hotel booking websites
Since the firmware is programmable, these shortcuts can be changed anytime.

## Tools Used

- KiCad 10
- Autodesk Fusion 360
- KMK/QMK Firmware

---

## What I Learned

While building this project I learned how to:

- Design a PCB from scratch
- Route traces and prepare Gerber files
- Add an OLED display to a custom PCB
- Design a 3D printable enclosure in Fusion 360
- Create a project around a specific theme instead of only focusing on hardware
